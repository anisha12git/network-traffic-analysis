# Network Traffic Analysis using Wireshark

## Objective
To capture and analyze real-time network traffic using Wireshark.

## Procedure
- Captured packets using Wireshark
- Generated traffic by browsing websites like Google and YouTube
- Applied filters such as `dns` to analyze specific protocols

## Observations
- DNS queries were observed for domains like google.com and pinterest.com
- DNS resolves domain names into IP addresses before communication
- Multiple packets are generated for a single request

## Tools Used
- Wireshark
